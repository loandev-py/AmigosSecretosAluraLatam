🎲 Amigo Secreto (Secret Friend Drawing App)
📝 Proyecto
Aplicación web para sortear amigos secretos de manera simple e interactiva.
✨ Características

Agregar amigos a la lista
Eliminar participantes
Validación de nombres
Sorteo aleatorio
Interfaz intuitiva

🛠 Tecnologías

HTML5
CSS3
JavaScript Vanilla

📦 Instalación
Requisitos

Navegador web moderno

Pasos

Clonar repositorio

bashCopygit clone https://github.com/usuario/amigo-secreto.git

Abrir index.html en navegador

🚀 Uso
Agregar Amigos

Escribe nombre
Pulsa botón "Añadir"

Eliminar Amigos

Haz clic en botón "✖"

Sortear

Pulsa "Sortear amigo"
Visualiza resultado aleatorio

🔒 Restricciones

Mínimo 3 amigos para sorteo
Sin nombres duplicados
